## Landing Page Best Practices

### Performance
- Use `next/image` for all images
- Lazy load below-fold components
- Keep hero section lightweight
- Preload critical fonts

### Conversion Optimization
- CTA above the fold
- Single clear action per section
- Social proof near forms
- Mobile-first form design

### Common Customization Points
- CSS variables in `globals.css` for quick theming
- `site.ts` and `brand.ts` for non-CSS config
- Component props for content changes
